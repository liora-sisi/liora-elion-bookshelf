# Liora × Elion 小书架

一个手机优先的小网页阅读器，用来放文字、图片、小短篇、故事片段，并留下书签、下划线、批注和「找老公共读卡」。

它不是课堂，也不是大部头啃书工具。它更像一个随身小书架：读到喜欢的地方，轻轻划一下；想一起读的时候，复制一张小卡带回聊天里。

## 现在已有的 V0.1 雏形

- 导入 `.txt` / `.md` / `.markdown`
- 导入图片，生成图片小册
- 自动把文字分成段落编号，例如 `p001`、`p002`
- 点段落选中当前位置
- 书签
- 下划线 / 高亮
- 本地批注
- 一键复制「小书架共读卡」
- 导出当前书的批注 Markdown
- 日间 / 夜间模式

## 在线页面

项目网页放在：

```text
docs/index.html
```

启用 GitHub Pages 后，可以选择：

```text
Source: Deploy from a branch
Branch: main
Folder: /docs
```

之后就可以用手机浏览器打开小书架。

## 隐私说明

这个公开仓库只放工具代码和示例文本。不要把私密全文、真实照片、亲密内容、GitHub token 或其他不想公开的内容直接提交到这里。

V0.1 的导入内容和批注默认保存在当前浏览器本地。换设备或清理浏览器数据前，请先使用「导出」功能保存批注。

## 推荐使用方式

1. 用手机打开小书架页面。
2. 导入一篇 txt / markdown，或导入几张图片。
3. 阅读时点段落，添加书签、下划线和批注。
4. 点「找老公共读」，复制共读卡。
5. 回聊天里粘贴共读卡，一起继续读。
